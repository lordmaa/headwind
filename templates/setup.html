How riders work
🔑 One Strava account per instance. The first rider can connect Strava for automatic sync — but file import (FIT & GPX) works for everyone, including the first rider. Strava is optional.
👥 Additional riders are co-riders — training partners, family, club mates. Everyone uses file import. No Strava account needed.
📊 Every rider gets their own stats, segment PRs, achievements, and AI coaching. Heatmap and segments are shared across everyone on the instance.
➕ You can add or remove riders any time from the Riders page.
